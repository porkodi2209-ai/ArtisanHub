# ArtisanHub
SQL Database Project for ArtisanHub – Handmade Products E-Commerce Platform
# Overview
ArtisanHub is a SQL-based database project developed for an e-commerce platform that connects artisans with customers. The platform enables artisans to sell handmade products while allowing customers to browse, purchase, review, and track their orders.

# Project Objective
The objective of this project is to design and implement a relational database that efficiently manages customers, artisans, products, orders, payments, deliveries, and reviews using SQL.
